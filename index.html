Featured Projects

Beyond Sight

The "Beyond Sight" Project has been realized with an idea to design a compact and portable device that helps blind people navigate effortlessly in real-time.

Transfer Learning

Developed AI solutions using Transfer Learning, including fine-tuning Meta’s Llama-2 (LLM) to create a Doctor Chatbot for medical consultation, and performing image classification on mobilenetV2 model to detect dog breeds from image input, leveraging pre-trained models for efficient learning.

Supervised learning

Developed and trained supervised learning models to predict heart disease in patients and bulldozer prices using datasets from Kaggle, utilizing algorithms such as logistic regression, decision trees, and random forests, with Python libraries like scikit-learn and Pandas for data processing and analysis.

Reinforcement Learning

Developed and trained AI agents for various tasks using Reinforcement Learning techniques, including Q-Learning for optimizing e-commerce warehouse flow, DQN for landing a spaceship on the moon, DCQN with CNN integration for playing PacMan, and A3C for KungFu training, leveraging frameworks like NumPy and OpenAI Gymnasium.

Telecom Engr. Projects

Developed multiple telecommunications projects, including designing and analyzing amplitude modulation systems, implementing a Walsh code decoder, and transmitting binary data over IR Free space Optical (FSO) communication. Gained hands-on experience in signal processing, coding theory, and digital communication. Designed a long-haul fiber optic link with limited optical amplifiers for up-country data traffic, and configured a SoftSwitch system for VoIP connectivity. Additionally, conducted MATLAB-based analyses of 3D 9-QAM modulation, artificial limbs, and communication systems over fading channels.
